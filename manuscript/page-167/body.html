Puis c'était le royaume d'argot: c'est-à-dire tous les voleurs de France, échelonnés par ordre de dignité; les moindres passant les premiers. Ainsi défilaient quatre par quatre, avec les divers insignes de leurs grades dans cette étrange faculté, la plupart éclopés, ceux-ci boiteux, ceux-là manchots, les courtauds de boutanche, les coquillarts, les hubins, les sabouleux, les calots, les francs-mitoux, les polissons, les piètres, les capons, les malingreux, les rifodés, les marcandiers, les narquois, les orphelins, les archisuppôts, les cagoux; dénombrement à fatiguer Homère.
